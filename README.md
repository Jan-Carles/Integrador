# Integrador

Dependências
Backend

npm install express cors jsonwebtoken bcrypt pg
npm install sequelize
npm install passport passport-jwt
npm install express-session passport-local

express → framework para criar servidor HTTP e rotas.

cors → permite que o frontend acesse o backend.

jsonwebtoken → gera e valida tokens JWT para autenticação.

bcrypt → criptografa senhas antes de salvar no banco.

pg → driver para conectar ao PostgreSQL.

sequelize → ORM para facilitar consultas SQL e manipulação de tabelas.

passport → biblioteca de autenticação flexível.

passport-jwt → estratégia do Passport para autenticação via JWT.

express-session → gerencia sessões de usuário no servidor.

passport-local → estratégia do Passport para login com usuário/senha.

Frontend
npm install @mui/material @mui/icons-material axios
npm install recharts
npm install chart.js react-chartjs-2

@mui/material → biblioteca de componentes visuais (botões, inputs, cards, etc.).

@mui/icons-material → ícones prontos para usar junto com Material UI.

axios → cliente HTTP para consumir o backend.

recharts → biblioteca de gráficos simples e responsivos.
